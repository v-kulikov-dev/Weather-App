### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/v-kulikov-dev/Weather-App.git
   ```

2. Go to folder
   ```sh
   cd Weather-App
   ```

2. Install NPM packages
   ```sh
   npm install
   ```

3. The app will open on port 3000
